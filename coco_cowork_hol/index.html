Step 1: Data Setup Manual

The only step requiring manual execution. Everything after this is done through Cortex Code prompts.

1.1 Create objects (SQL cell)

Open a Snowflake Workspace and paste Setup/setup.sql into a SQL cell. This creates:

  • Warehouse HOL_WH (XS, auto-suspend 60s)
  • Database HOL_COCO_CWORK with schemas: DATA, TOOLS, AGENTS
  • Three tables with UUID primary keys
  • Internal stage HOL_STAGE for data
  • Skills stage SKILLS_STAGE in TOOLS schema

1.2 Upload CSV files and skills (Python cell)

# Run in a Python cell in your Workspace
from snowflake.snowpark.context import get_active_session
session = get_active_session()

# Upload data files
session.file.put("data/dim_store.csv", "@HOL_STAGE/dim_store", auto_compress=True, overwrite=True)
session.file.put("data/dim_item.csv", "@HOL_STAGE/dim_item", auto_compress=True, overwrite=True)
session.file.put("data/fact_item_sales.csv", "@HOL_STAGE/fact_item_sales", auto_compress=True, overwrite=True)

# Upload agent skills
session.sql("USE SCHEMA TOOLS").collect()
session.file.put("../Skills/anomaly_detection/SKILL.md", "@SKILLS_STAGE/skills/anomaly_detection/", auto_compress=False, overwrite=True)
session.file.put("../Skills/sales_report/SKILL.md", "@SKILLS_STAGE/skills/sales_report/", auto_compress=False, overwrite=True)
session.sql("USE SCHEMA DATA").collect()

1.3 Load data (SQL cell)

Run the COPY INTO statements (steps 7-8 in setup.sql) to load dimensions and facts.

Step 2: Create Semantic View Cortex Code

Open Prompts/semantic_view.md in Cortex Code and run:

/semantic-view @semantic_view.md

What CoCo does

  • Discovers all tables in HOL_COCO_CWORK.DATA
  • Runs FastGen to generate dimensions, facts, and relationships
  • Creates 8 verified queries (VQRs) for sales analytics
  • Validates the YAML against Snowflake
  • Deploys to HOL_COCO_CWORK.TOOLS

Verified Queries Included

#Question
1What is the total revenue?
2What is the total revenue by category?
3What is the total revenue by state?
4What are the top 10 stores by revenue?
5What are the top 10 best selling items by quantity?
6What is the monthly revenue trend?
7What is the average discount percentage by category?
8How many total transactions are there?
check_circle
Result: HOL_COCO_CWORK.TOOLS.HOT_FOOD_SALES_ANALYTICS — 3 tables, 2 relationships, 8 VQRs

Step 3: Create Cortex Agent Cortex Code

Open Prompts/agent.md in Cortex Code and run:

/cortex-agent @agent.md

What CoCo does

  • Creates agent workspace and spec
  • Configures orchestration instructions (role, boundaries, business rules)
  • Configures response instructions (assertive tone, multilingual)
  • Attaches semantic view as a cortex_analyst_text_to_sql tool
  • Attaches 2 skills from @SKILLS_STAGE
  • Deploys to HOL_COCO_CWORK.AGENTS

Agent Capabilities

Tools & Skills

  • tool Text-to-SQL via semantic view
  • skill Anomaly detection
  • skill Sales report generator

Behavior

  • Assertive, no hedging
  • Multilingual responses
  • Asks clarifying follow-ups
  • Tables and charts
check_circle
Result: HOL_COCO_CWORK.AGENTS.HOT_FOOD_SALES_AGENT — ready with 1 tool + 2 skills

Step 4: Run Evaluations Cortex Code

Attach Prompts/evaluations.md in Cortex Code:

@evaluations.md

What CoCo does

  • Queries underlying tables to build ground truth answers
  • Creates evaluation dataset with 10 questions across 5 categories
  • Registers dataset with Snowflake's evaluation framework
  • Runs evaluation measuring answer_correctness and logical_consistency
  • Presents results with per-question scores

Agent Skills

The agent includes two server-side skills stored on @HOL_COCO_CWORK.TOOLS.SKILLS_STAGE. Skills are SKILL.md files that give the agent specialized workflows.

Skill Anomaly Detection

Identifies statistically unusual sales patterns using z-score analysis over a 7-day rolling window.

  • Dimensions: Store, Category, State, Item
  • Metrics: Revenue, Quantity, Transactions
  • Output: Spikes and drops with z-scores

"Find any anomalies in revenue by category"

Skill Sales Report

Generates a structured executive report with top products, trends, missed opportunities, and actions.

  • Scope: By Store or By State
  • Sections: Summary, Top 10, Trends, Gaps, Actions
  • Output: Full executive report

"Generate a sales report for New York"
